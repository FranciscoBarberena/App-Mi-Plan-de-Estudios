# Mi Plan de Estudios 

* Una aplicación Android que sirve para llevar un registro de las materias aprobadas de tu carrera de la UNLP.

## [Descargar APK de la útlima versión](https://github.com/FranciscoBarberena/Mi-Plan-de-Estudios/releases/latest)


## Funcionalidades
* Seguimiento de Materias
  * Marca materias como aprobadas o cursadas.
* Sistema de Correlatividades
  * Bloquea materias si no cumplís los requisitos para cursar/rendir.
* Promedio y porcentaje de la carrera completado
  * Calcula tu promedio actual, y el porcentaje de materias que ya aprobaste.
* Copia de Seguridad 
  * Exporta e importa tu progreso en formato JSON.
* Personalización
  * Modo oscuro
  * Colores de acento
* 26 carreras de la UNLP!

## Tecnologías
* Kotlin
* Jetpack Compose (UI)
* Room Database (Persistencia)
* ViewModel & StateFlow (Arquitectura MVVM)
* Gson (Backup de datos)
